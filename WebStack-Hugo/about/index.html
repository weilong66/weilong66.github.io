一个基于 Hugo 的静态响应式网址导航主题

本项目是基于纯静态的网址导航网站 webstack.cc 制作的 Hugo 主题，是一个基于 Hugo 的静态响应式网址导航主题。

说明

该导航页基于 shenweiyan/WebStack-Hugo 这个版本修改而来。

主题地址

  • WebStack-Hugo-WL

主题演示地址

  • 站点：https://weilong66.github.io/WebStack-Hugo/
  • 源码：https://github.com/weilong66/weilong66.github.io

特色功能

这是 Hugo 版 WebStack 主题。可以借助下面的平台直接托管部署，无需服务器。

  • Webify | Netlify | Cloudflare Pages | Vercel | Github Pages

特点：

  • 采用 Hugo 部署方式，方便高效。
  • 主要的配置信息都集成到了 config.toml，一键完成各种自定义的配置。
  • 导航的各个信息都集成在 data/webstack.yml 文件中，方便后续增删改动。 
    - taxonomy: 常用网站                        # 左侧导航条顶级菜单名称。
  icon: far fa-star                        # 左侧导航条顶级菜单最左侧Font Awesome图标。
  list:                                
    - term: 常用网站                        # 导航条子菜单名称。
      links:                           
        - title: 邮箱                       # 网站名称
          logo: 邮箱.png                    # 网站Logo
          url: http://mail.email.com/       # 网站URL
          description: 邮箱网站。           # 网站介绍
  • 支持手机电脑自适应以及夜间模式。
  • 支持搜索收录的站点
  • 支持调用互联网搜索功能

功能

  • 添加 一言API。
  • 添加 和风天气 插件。
    • 因和风天气官方插件失效，这里改为使用自己粗略制作的天气插件。
  • 添加 天气+ 插件。（与 和风天气 插件二选一）
  • 添加小飞机彩蛋和仓鼠宠物插件。
  • 添加本地小游戏站点。
